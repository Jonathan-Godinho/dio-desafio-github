# Anotações:

## Array 
#### **Vetores:** 
- Array  
- Única dimensão 
- Tamanho fixo

# Exemplo:

Índices  | 0  |1   |2   |3   |4   |5   |
-------- | ---|----|----|----|----|----|    
Valores  | 10 | 14 | 20 | 09 | 16 | 22 |

----------------------------------------

#### **Matriz:** 
- Array 
- Tamanho fixo 
- Multidimensionais

# Exemplo:

--|  0 |  1 |  2 |   3 |
-----|----|---|---|---|
0    |(0,0)|(0,1)| (0,2)|(0,3)|
1    |(1,0)|(1,1)| (1,2)|(1,3)|
2    |(2,0)|(2,1)| (2,2)|(2,3)|
3    |(3,0)|(3,1)| (3,2)|(3,3)|

----------------------------------------
